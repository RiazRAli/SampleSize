🎯

Confidence Level

The probability (e.g. 95%) that the method captures the true parameter. Driven by the Z-score. Higher confidence = wider interval.

📊

Sample Size (n)

Number of observations. We divide by √n — more data = smaller standard error = narrower interval. But 4× the data only halves the width.

↔️

Margin of Error (MOE)

The "±" value: Z × σ/√n. How far the interval extends each side of x̄. This is what we want to control in study design.

The Core Equation MOE = Zα/2 × σ / √n

Key Takeaways
What to Remember

The essential relationships.

🎯

↑ Confidence Level → ↑ Width

Higher confidence demands a larger Z-score. If n is fixed, the margin of error must grow. Going from 90% to 99% nearly doubles the interval.

📊

↑ Sample Size → ↓ Width

More data shrinks standard error (σ/√n). Square root relationship means diminishing returns — 4× the data for half the width.

🔒

Fix MOE, ↑ CL → Much ↑ n

Want a tight interval AND high confidence? You'll need a much larger sample. This is the core study design tradeoff.

🔄

"95% Confident" = Repeated Sampling

The true mean is fixed. 95% confidence means 95% of intervals built this way will capture it. It describes the method, not this one interval.
